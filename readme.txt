== Description ==
If you need a short url of a set of plugin and if anyone visit to that url then you need to send theme to random url then this is the plugin you are finding for.

You will be able to import CSV file with a url and priority and after importing it will give you a shorturl.

== Installation ==
Just install the plugin you will get a admin menu called "URL shorter" here you will be able to import csv file. CSV file format should be First column url and second column priority value (must be numaric)
